# lab-pc-sports

这是一个用于归档和发布独立 HTML 页面的仓库。仓库中的页面均为独立构建，不依赖特定域名或外部服务，适合本地浏览或静态部署。

## 项目简介

本仓库主要用于存放与 PC 端运动相关主题的 HTML 页面。每个页面内容独立，结构清晰，便于查看、复用或二次修改。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── img/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面存放于此，每个页面为一个完整功能单元。
- `assets/`：页面间共享的静态资源，按类型分目录存放。

## 页面归档说明

- 每个页面均为独立 HTML 文件，无跨页面依赖。
- 页面中使用的资源优先引用 `assets/` 目录下的文件，或使用内联方式嵌入。
- 页面内容不指向任何具体网站或服务，仅为功能演示或技术展示。

## 维护说明

- 新增页面：在 `pages/` 下创建新的 HTML 文件，并确保资源引用正确。
- 修改页面：直接编辑对应 HTML 文件，保持与 `assets/` 中资源的同步。
- 删除页面：移除对应文件，并检查是否有引用关系需要清理。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中打开 `pages/` 目录下任意 HTML 文件，即可查看对应页面。
3. 也可将整个仓库部署到任意静态服务器，通过目录路径访问各页面。
